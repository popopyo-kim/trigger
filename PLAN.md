# Streamlit 이미지 생성기 - 기획서

## 개요
Streamlit 기반의 "이미지 생성기" 앱을 `app.py` 단일 파일로 개발한다.
대본을 입력하면 초단위로 분할 → 이미지 프롬프트 생성 → 이미지 생성까지 자동화하는 워크플로우.

---

## 전체 워크플로우

```
[1] 대본 입력
    ├─ 도입부: 시간 설정 (1~6초, 1초 단위) → 자동 분할 → 수동 분할/편집/병합
    └─ 본문:  시간 설정 (5~30초, 5초 단위) → 자동 분할 → 텍스트 편집

[2] 분할 결과 최종 확인 (도입부 글자수/초수 경고 포함)

[3] 프롬프트 생성 (Gemini 2.5 Flash) + 언어 선택 + 형식 프롬프트 편집

[4] 이미지 생성 (Gemini API) + 다운로드
```

---

## 1단계: 대본 입력

### 도입부
- `text_area`로 도입부 대본 입력
- 시간 설정: 1~6초, 1초 단위 슬라이더
- 자동 분할 버튼 → 글자 수 기준(1초당 4~5글자)으로 자동 분할

### 본문
- `text_area`로 본문 대본 입력
- 시간 설정: 5~30초, 5초 단위 슬라이더 (5, 10, 15, 20, 25, 30초)
- 자동 분할 버튼 → 글자 수 기준으로 자동 분할

---

## 2단계: 초단위 분할 결과 확인/편집

### 도입부 - 자동 + 수동 하이브리드

자동 분할 후 다음 수동 편집 기능 제공:

| 기능 | 설명 |
|------|------|
| **분할** | 선택한 컷의 텍스트를 지정 위치에서 2개로 나눔 |
| **삭제** | 해당 컷 삭제 |
| **+ 컷 추가** | 빈 컷을 수동으로 추가 |
| **병합** | 인접한 2개 컷을 하나로 합침 |
| **텍스트 직접 편집** | 각 컷의 텍스트를 자유롭게 수정 |
| **글자수 / 예상 시간** | 각 컷 옆에 글자수와 예상 초수 실시간 표시 |

### 6초 초과 경고
- 각 컷의 글자 수가 6초 분량(약 27~30글자)을 초과하면 경고 표시
- 분할을 권장하는 안내 메시지 출력

### 본문
- 자동 분할 결과를 텍스트 편집으로 수정 가능
- 수동 분할 기능 불필요

### 출력 구조 예시
```
📌 도입부 (3초 기준)
  1. 여러분 오늘 놀라운 이야기를    [12글자/약3초] ✂️분할 🗑삭제
  2. 지금부터 시작합니다            [9글자/약2초]  ✂️분할 🗑삭제
  [+ 컷 추가]

📌 본문 (10초 기준)
  3. 부자들은 위기를 기회로 삼습니다...
  4. 주식 시장이 폭락할 때 오히려...
```

## 3단계: 이미지 프롬프트 생성

### LLM 설정
- Gemini API 사용 (google-generativeai SDK)
- 기본 모델: `gemini-2.5-flash-preview-05-20`
- 사이드바에서 모델명 변경 가능

### 형식 프롬프트 (System Prompt)
- 사용자가 직접 입력/수정 가능한 형식 프롬프트 영역 제공
- 기본값으로 아래 Gems 시스템 지침 미리 채움

### 기본 형식 프롬프트 (Gems System Instructions v7.0)

```
당신은 '2D 스틱맨 애니메이션 전문 프롬프트 디렉터'입니다.

스타일 가이드:
- 캐릭터: Pure-white round faces, single hard cel shading, thick black outline,
  thicker torso and neck, stick limbs, flat matte colors.
- 배경: 저채도 평면 블록, 글자 절대 금지.
- 네거티브: 3D, photoreal, gradient, soft light, text, letters, speech bubble.

장면 해석:
- 감정은 눈썹/입선으로, 동작은 명확한 동사로 표현
- 추상적 개념은 인물+아이콘/도형으로 변환
- 모든 간판, 화면, 문서에 글자 대신 기호/도형만 사용

출력 템플릿:
> Upgraded stick-man 2D with thick black outline, pure white faces,
> single hard cel shading, thicker torso and neck, flat matte colors;
> SCENE: [행동 및 아이콘 묘사 (영문) + no text/letters 강조]
```

### 언어 선택
- 한국어 / 일본어 / 영어 / 언어없음
- 선택한 언어에 따라 프롬프트에 언어 지시 포함

### 출력
- 각 세그먼트별 영문 이미지 프롬프트를 CSS 코드 블록으로 표시
- 각 프롬프트 수동 편집 가능

---

## 4단계: 이미지 생성

### 이미지 생성 API
- Gemini API 사용 (프롬프트 생성과 동일 API Key 공유)
- 모델명은 사이드바에서 자유 입력 (나노바나나2 등)

### 기능
- 각 프롬프트별 이미지 1장 생성
- 생성된 이미지를 순서대로 표시 (도입부/본문 구분 유지)
- 개별 다운로드 + 전체 ZIP 다운로드

---

## 사이드바 구성

| 구분 | 항목 | 기본값 |
|------|------|--------|
| **API 설정** | Gemini API Key | (입력) |
| **프롬프트 생성** | 모델명 | `gemini-2.5-flash-preview-05-20` |
| **이미지 생성** | 모델명 | (입력) |
| | 이미지 사이즈 | 선택 |
| **공통** | 언어 선택 | 한국어/일본어/영어/언어없음 |

API Key는 하나로 공유하고, 모델명만 각각 지정하는 방식.

---

## 기술 스택

- **Streamlit**: UI 전체
- **google-generativeai**: Gemini API (프롬프트 생성 + 이미지 생성)
- **zipfile / io**: ZIP 다운로드 생성
- **st.session_state**: 단계별 상태 관리
